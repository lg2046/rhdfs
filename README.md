# Rhdfs

hdfs 客户端 基于jruby

## Installation

Add this line to your application's Gemfile:

    gem 'rhdfs'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rhdfs

## Usage

   自带的jar为hadoop1.2.1的jar
   如果要使用其他版本，把lib/jars里面的替换为相应的hadoop的就可以了(包含hadoop目录下与hadoop/lib下面的jar一起拷贝过来就可以了)
